# Αρχιτεκτονική Εφαρμογής: AI Stock Analyzer Pro (Desktop Edition)

Το **AI Stock Analyzer** είναι ένα εργαλείο που διευκολύνει την ανάλυση μετοχών με τη χρήση Τεχνητής Νοημοσύνης (Cloud και Τοπικά Μοντέλα). Αντλεί κείμενο από πηγές (URLs) που ορίζει ο χρήστης, καθώς και χρηματιστηριακά δεδομένα, και τα τροφοδοτεί σε AI μοντέλα για την παραγωγή μιας ολοκληρωμένης σύνοψης και οικονομικής ανάλυσης. Η συγκεκριμένη έκδοση είναι σχεδιασμένη ως αυτόνομη Desktop εφαρμογή για Windows.

## Βασικά Στοιχεία Εφαρμογής (Components)

### Τι νέο υπάρχει στην έκδοση 1.4

- **Εξαγωγή σε PDF**: Προσθήκη δυνατότητας απευθείας μετατροπής της αναφοράς AI σε αρχείο `.pdf` χωρίς να χάνεται η μορφοποίηση.
- **Δυναμικό μέγεθος γραμματοσειράς**: Νέα κουμπιά αυξομείωσης (+/-) στην τελική αναφορά AI για καλύτερη αναγνωσιμότητα.
- **Βελτιώσεις Διεπαφής**: Ορισμός της γραμματοσειράς `Segoe UI` ως προεπιλογή για καλύτερη απεικόνιση των Ελληνικών χαρακτήρων, γαλάζιες επικεφαλίδες στα Accordion και εργονομική σύμπτυξη των δευτερευόντων πλαισίων (Collapsed by default).

### 1. `desktop_app.py` (CustomTkinter GUI)

Η κύρια, αυτόνομη έκδοση της εφαρμογής που βασίζεται στο framework του CustomTkinter για τη δημιουργία σύγχρονου παραθυρικού περιβάλλοντος.

- **Διεπαφή Χρήστη (UI):** Προσφέρει πλευρική μπάρα (sidebar) με επιλογή Γλώσσας (Ελληνικά/Αγγλικά), δυνατότητα απόκρυψης (Focus Mode / ☰), έλεγχο "Δημιουργικότητας" (Temperature Slider) για το AI, και ενσωματωμένα κουμπιά επικόλλησης. Υποστηρίζει την προσθήκη, πλήρη αναδιάταξη (sorting up/down), επεξεργασία της Watchlist και προσωποποιημένες **Σημειώσεις (Notes)** ανά τίτλο. Το κεντρικό UI χρησιμοποιεί ανεξάρτητη κύλιση (split-scroll) ανάμεσα στη στήλη δεδομένων και στη στήλη της επισκόπησης/ανάλυσης (με έξυπνη δυναμική εμφάνιση/απόκρυψη των ενοτήτων API).
- **Διαχείριση Τοπικών Αρχείων & Άρθρων:** Επιτρέπει τη φόρτωση τοπικών αρχείων PDF (μέσω `PyPDF2`) και TXT, καθώς και την απευθείας επικόλληση κειμένου άρθρων σε 3 ξεχωριστά Textboxes, ενσωματώνοντας το περιεχόμενό τους στο πλαίσιο του AI.
- **Διαχείριση Πηγών (Tabs):** Ενσωματωμένο Tabview για δυναμική εμφάνιση των γραφημάτων, των τελευταίων ειδήσεων (DuckDuckGo) και των ειδήσεων του NewsAPI.org. Υποστηρίζει ενσωμάτωση του εταιρικού ιστότοπου και προσθήκη Custom URLs τα οποία το σύστημα "διαβάζει" αυτόματα στο παρασκήνιο (Web Scraping). Οι κλήσεις στα API ελέγχονται από έξυπνους μετρητές ημερήσιας χρήσης.
- **RSS Feeds & Custom URLs:** Δυνατότητα προσθήκης, αναδιάταξης και αυτόματης (silent) αποθήκευσης URLs/Feeds. Στα RSS υποστηρίζεται προηγμένο φιλτράρισμα λέξεων (με χρήση τελεστών `+` και `,`), δυναμική επισήμανση (highlighting) στο UI και φιλτράρισμα ανά ημερομηνία για προβολή άρθρων σε ξεχωριστό Tab.
- **Ιστορικό & Δεδομένα:** Δυνατότητα αποθήκευσης, επαναφόρτωσης και χειροκίνητης διαγραφής των 10 πιο πρόσφατων αναλύσεων. Παρέχεται επίσης πλήκτρο "Εκκαθάρισης Όλων των Δεδομένων" (επαναφορά εργοστασιακών ρυθμίσεων).
- **Αυτόματος Έλεγχος Ενημερώσεων:** Ενσωματωμένος μηχανισμός (background thread) που επικοινωνεί με το GitHub API κατά την εκκίνηση για να ειδοποιήσει τον χρήστη όταν υπάρχει νέα διαθέσιμη έκδοση (Release).
- **Βελτιστοποίηση Απόδοσης (Lazy Loading):** Οι βαριές βιβλιοθήκες (π.χ. `matplotlib`, `yfinance`, `pandas`) φορτώνονται δυναμικά μόνο όταν είναι απολύτως απαραίτητες, εξασφαλίζοντας την ακαριαία εκκίνηση του γραφικού περιβάλλοντος.
- **Διαχείριση Δεδομένων:** Χρησιμοποιεί ένα τοπικό αρχείο `user_data.json` για τη μόνιμη αποθήκευση των ρυθμίσεων. Αποθηκεύει API keys, τη watchlist, τα URLs, το ιστορικό αναλύσεων (`history`), και την ημερήσια χρήση των APIs (`api_usage`).
- **Χρηματιστηριακή Απεικόνιση & Γραφήματα:** Ενσωματώνει πολλαπλές πηγές για την εμφάνιση δεδομένων:
  - **Yahoo Finance (`yfinance`):** Για ιστορικά δεδομένα, δυναμικό υπολογισμό Τάσης μέσω SMA, ενδείξεις Υπεραγόρασης/Υπερπώλησης στο RSI, καθώς και άντληση δεικτών **Οικονομικής Υγείας** (Revenue Growth, ROE, Free Cash Flow κ.ά.). Το γράφημα υλοποιείται μέσω `matplotlib` με **διάταξη 4 επιπέδων (Subplots)**: Απομονωμένο λεπτό Hover Track στην κορυφή, Διαγράμματα κεριών (με αυστηρά όρια Y-axis για αποφυγή συμπίεσης), Όγκος συναλλαγών (Volume), και RSI.
  - **Financial Times & Investing.com:** Για ανάκτηση εναλλακτικών τιμών σε πραγματικό χρόνο.
  - **Alpha Vantage & Finnhub:** Για on-demand άντληση εκτενών θεμελιωδών δεδομένων τα οποία εμφανίζονται σε πραγματικό χρόνο στο UI πριν την ανάλυση.
  - **NewsAPI.org:** Ενσωμάτωση ειδήσεων παγκόσμιας εμβέλειας, με προηγμένη υποστήριξη λογικών τελεστών (`+` / `,`) για το φιλτράρισμα λέξεων, και αυτόματο highlighting των όρων αναζήτησης στο UI.
- **Ασύγχρονη Εκτέλεση (Threading):** Οι βαριές διεργασίες (λήψη δεδομένων, επικοινωνία με το AI) εκτελούνται σε ξεχωριστά threads για να μην "παγώνει" το γραφικό περιβάλλον κατά την αναμονή.
- **Έξυπνη Εξαγωγή (Word Export):** Η εξαγωγή της ανάλυσης σε `.docx` διατηρεί τη μορφοποίηση έντονου κειμένου (Markdown Bold) και μετατρέπει αυτόματα τα URLs σε ενεργά (clickable) hyperlinks μέσω XML OxmlElements.

### 2. `user_data.json` (Τοπική Βάση)

Ένα απλό JSON αρχείο που λειτουργεί ως βάση δεδομένων, διατηρώντας την κατάσταση της εφαρμογής μεταξύ των συνεδριών (Watchlist, APIs, επιλεγμένα URLs, Ιστορικό Αναλύσεων). Περιλαμβάνει μηχανισμό αυτόματης διόρθωσης σφαλμάτων (data migration) κατά την εκκίνηση.

### 3. `stock_fetcher.py` (Μηχανισμός Δεδομένων)

Αναλαμβάνει όλη την επικοινωνία με τον "έξω κόσμο". Φροντίζει για το κατέβασμα των τιμών και δεικτών υγείας από το yfinance **(αναγνωρίζοντας έξυπνα αν πρόκειται για Μετοχή, ETF ή Χρηματιστηριακό Δείκτη)**, τον υπολογισμό των τεχνικών δεικτών με το `pandas-ta`, το scraping τιμών (FT, Investing.com) μέσω `cloudscraper` / `BeautifulSoup`, την "καθαρή" εξαγωγή κειμένου άρθρων μέσω `trafilatura`, την εύρεση ειδήσεων με το `ddgs`, τις απευθείας κλήσεις REST API (με μετάφραση queries σε NewsAPI), καθώς και την ανάλυση συνθέτων RSS feeds (με custom parsing ημερομηνιών/λέξεων) μέσω της `feedparser`. Ενσωματώνει έξυπνους ελέγχους (heuristics) κατά τη μετατροπή των δεδομένων, αναγνωρίζοντας πότε τα API επιστρέφουν τα ποσοστά (π.χ. μερίσματα, ROE) ως ακέραιους αριθμούς αντί για δεκαδικούς, αποτρέποντας την εμφάνιση λανθασμένων εξωπραγματικών τιμών (π.χ. 1500%).

### 4. `ai_service.py` (Διαχείριση Τεχνητής Νοημοσύνης)

Απομονώνει τη λογική επικοινωνίας με τα Μεγάλα Γλωσσικά Μοντέλα (LLMs). Παρέχει ενιαίες συναρτήσεις για την ανάκτηση των διαθέσιμων μοντέλων και την αποστολή του Prompt, γεφυρώνοντας τις διαφορές μεταξύ του Cloud API του Google Gemini (`google.genai`) και του τοπικού εξυπηρετητή του Ollama. Παράλληλα ενσωματώνει την παράμετρο Temperature.

### 5. `data_manager.py` (Διαχείριση Μνήμης & Αποθήκευσης)

Υπεύθυνο για την ασφαλή ανάγνωση (load) και εγγραφή (save) του αρχείου `user_data.json`. Περιλαμβάνει try-except blocks με logging (καταγραφή σφαλμάτων στο `app_errors.log`) σε περίπτωση αλλοίωσης του αρχείου, επιστρέφοντας ένα καθαρό template για να μην καταρρεύσει η εφαρμογή.

### 6. `document_exporter.py` (Εξαγωγή Αναφορών)

Χρησιμοποιεί τη βιβλιοθήκη `python-docx` για να μετατρέπει τη ροή κειμένου (markdown) του AI σε επαγγελματικό αρχείο Microsoft Word, επιτρέποντας στον χρήστη να αποθηκεύσει την ανάλυση για παρουσίαση ή αρχειοθέτηση.

### 7. `translations.py` (Πολυγλωσσικό Σύστημα - i18n)

Περιέχει ένα δομημένο λεξικό (`TRANSLATIONS`) με όλα τα κείμενα του περιβάλλοντος χρήστη, τα tooltips και τα System Prompts του AI. Επιτρέπει την πλήρη μετάφραση της εφαρμογής (προς το παρόν σε Ελληνικά και Αγγλικά).

## Ροή Δεδομένων (Data Flow)

1. **Επιλογή & Εισαγωγή:** Ο χρήστης επιλέγει μια μετοχή από το Dropdown μενού της watchlist.
   *(Παράλληλα με την εκκίνηση, εκτελείται αθόρυβος έλεγχος εκδόσεων - Update Check - στο παρασκήνιο).*
2. **Ανάκτηση Δεδομένων (Data Fetching):**
    - **Scraping Τιμών:** Αντλούνται οι τρέχουσες τιμές από FT και Investing.com.
    - **Yahoo Finance:** Το `yfinance` αντλεί ιστορικά δεδομένα, αναγνωρίζει τον τύπο του τίτλου (Equity/ETF/Index), υπολογίζει τεχνικούς δείκτες (`pandas-ta`) και επιστρέφει βασικά στατιστικά με προσαρμοσμένο AI Context.
    - **On-Demand APIs:** Εάν οριστούν τα κλειδιά και επιλεγούν τα Checkboxes, γίνονται κλήσεις στα API των Alpha Vantage και Finnhub σε πραγματικό χρόνο.
    - **NewsAPI API:** Εάν επιλεγεί, αντλεί άρθρα (με δυνατότητα φιλτραρίσματος επιπρόσθετης λέξης-κλειδιού ή γλώσσας).
    - **Ειδήσεις:** Με την επιλογή της μετοχής, γίνεται αυτόματη αναζήτηση των πιο πρόσφατων ειδήσεων της χρονιάς (έως 1 έτος) μέσω `ddgs`.
    - **Εταιρικό Site, Custom URLs & RSS Feeds:** Το εταιρικό site αντλείται αυτόματα (εφόσον είναι τσεκαρισμένο), ενώ τα αποθηκευμένα URLs και τα επιλεγμένα άρθρα από RSS Feeds διαβάζονται στο παρασκήνιο και εφαρμόζονται φίλτρα ημερομηνίας/λέξης.
    - **Τοπικά Αρχεία & Επικολήσεις:** Τα επισυναπτόμενα αρχεία PDF/TXT αναλύονται για εξαγωγή κειμένου, και τα επικολλημένα άρθρα προετοιμάζονται.
3. **Δόμηση Πλαισίου (Context Building):** Όλες οι πληροφορίες (στατιστικά, API data, τεχνικοί δείκτες, επιλεγμένες ειδήσεις, έγγραφα, και scraped text από URLs) συνδυάζονται δυναμικά σε ένα δομημένο ενιαίο κείμενο (Prompt).
4. **Επεξεργασία μέσω Τεχνητής Νοημοσύνης:**
   - Εφαρμόζεται το επιλεγμένο `Temperature` (0.0 έως 1.0) για έλεγχο δημιουργικότητας.
   - Εάν επιλεγεί το **Gemini (Cloud)**: Η εφαρμογή επικοινωνεί με το Google AI Studio (μέσω `google.genai`) και στέλνει το prompt μαζί με το κείμενο που συλλέχθηκε.
   - Εάν επιλεγεί το **Ollama (Τοπικά)**: Το αίτημα δρομολογείται στο τοπικό daemon του Ollama.
5. **Παρουσίαση & Καταγραφή:** Η τελική ανάλυση επιστρέφεται από το ασύγχρονο thread, με προσαρτημένη στο τέλος τη λίστα των πηγών που χρησιμοποιήθηκαν (URLs και ειδήσεις), και εμφανίζεται απευθείας στο TextBox. Παράλληλα αποθηκεύεται δυναμικά στο τοπικό Ιστορικό της εφαρμογής (έως 10 εγγραφές).

## Βασικές Βιβλιοθήκες (Dependencies)

Οι παρακάτω βιβλιοθήκες αποτελούν τον πυρήνα του συστήματος:

| Βιβλιοθήκη | Χρήση |
| :--- | :--- |
| `customtkinter` | Δημιουργία του σύγχρονου Desktop GUI. |
| `pyinstaller` | Δημιουργία εκτελέσιμου. Διανομή σε μορφή φακέλου (χωρίς `--onefile`) για μηδενικό χρόνο αποσυμπίεσης κατά την εκκίνηση. |
| `requests` | Αποστολή HTTP requests για APIs και web scraping. |
| `cloudscraper` | Εξειδικευμένη βιβλιοθήκη για παράκαμψη του Cloudflare. |
| `beautifulsoup4` | Αναγνώριση και εξαγωγή του κειμένου μέσα από το DOM της HTML. |
| `trafilatura` | Προηγμένη εξαγωγή καθαρού κειμένου (χωρίς μενού/διαφημίσεις) από ιστοσελίδες και URLs. |
| `google-genai` | Επίσημο SDK για επικοινωνία με τα μοντέλα Google Gemini. |
| `ollama` | Python client για σύνδεση με το τοπικό Ollama server. |
| `yfinance` | Ανάκτηση δεδομένων τιμών και μετοχών από το Yahoo Finance. |
| `pandas-ta` | Υπολογισμός τεχνικών δεικτών (RSI, MACD, SMA) πάνω στα δεδομένα του yfinance. |
| `ddgs` | Εκτέλεση αναζητήσεων ειδήσεων (ανεξαρτήτως γλώσσας) μέσω DuckDuckGo. |
| `matplotlib` | Δημιουργία και απεικόνιση των χρηματιστηριακών διαγραμμάτων (Candlesticks, Όγκος, SMAs, RSI). |
| `PyPDF2` | Ασφαλής εξαγωγή και ανάγνωση κειμένου από τοπικά αρχεία μορφής PDF. |
| `python-docx` | Προγραμματιστική δημιουργία MS Word `.docx` εγγράφων. |
| `feedparser` | Ανάγνωση και ανάλυση δομών RSS/Atom feeds. |
| `python-dateutil` | Ευέλικτη ανάλυση ημερομηνιών για το φιλτράρισμα των RSS άρθρων. |
